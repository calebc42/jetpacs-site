# Jetpacs Composer

> No-code CRUD apps for Jetpacs, described as a single declarative org document — no code in the app document, ever.

Jetpacs Composer builds Tier-1 <a href="https://github.com/calebc42/jetpacs">Jetpacs</a> apps as declarative org documents. It is the AppSheet model, org-native: where AppSheet builds an app around a spreadsheet, Composer builds one around org tables. A single `app.org` file describes the whole app — its views, datasources, and column types — and it runs on the phone as a native-UI Jetpacs app, with org files as the backend and Emacs as the brain.
